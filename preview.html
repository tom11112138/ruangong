如何启动完整项目

由于当前系统未安装Node.js，无法启动完整服务。请按以下步骤操作：

1. 安装Node.js
访问 https://nodejs.org 下载并安装Node.js (建议版本16+)
2. 安装项目依赖
# 后端依赖
cd backend
npm install

# 前端依赖
cd ../frontend
npm install
3. 启动项目
# 启动后端服务 (端口3001)
cd backend
npm start

# 启动前端服务 (端口3000)
cd ../frontend
npm run dev

安装完成后，访问 http://localhost:3000 即可查看完整项目。
